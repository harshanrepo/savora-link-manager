## Installation
1. Download or clone this repository
2. Open Chrome/Brave and go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select the Savora folder

## Usage
- **Save Input**: Paste a URL and click "SAVE"
- **Save Tab**: Click "SAVE TAB" to save current page
- **Delete All**: Double-click "DELETE ALL"
